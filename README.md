## How to Use This Code as a Submodule

Add the submodule:

```
git submodule add https://github.com/retojs/comicvm-geometry-2d.git
```

Update submodule changes:

``` 
cd comicvm-dom
git pull
```

see: https://git-scm.com/book/en/v2/Git-Tools-Submodules
